4dgs-slam-ros2
===============

A complete ROS2 package for the 4D Gaussian Splatting SLAM system.

## Key Features

- **Full ROS2 Integration**: Complete node implementation with proper ROS2 standards
- **Real-time Performance**: Optimized for real-time SLAM processing
- **Dynamic Scene Reconstruction**: Handles static and dynamic environments
- **Multiple Launch Options**: Basic, bag playback, and visualization configurations
- **Comprehensive Documentation**: Extensive guides and examples
- **Production Ready**: Designed for real-world robotics applications

## Quick Setup

```bash
# Install the package
cd ~/dev
git clone https://github.com/yourusername/4dgs-slam-ros2.git
cd 4dgs-slam-ros2

# Run setup script
python3 scripts/install.py --pretrained

# Run with a bag file
ros2 run 4dgs_slam 4dgs_slam_node
```

## Installation

### System Requirements

- Ubuntu 20.04/22.04
- ROS 2 Humble or Rolling
- NVIDIA GPU with CUDA 11.x
- Python 3.8+

### Quick Install

```bash
cd ~/dev/4dgs-slam-ros2
python3 scripts/install.py --pretrained
```

## Usage

### Basic Usage

```bash
ros2 run 4dgs_slam 4dgs_slam_node
```

### With Bag File

```bash
ros2 run 4dgs_slam 4dgs_slam_node
# In another terminal: ros2 bag play your_bag.bag --clock
```

### With Visualization

```bash
ros2 launch 4dgs_slam 4dgs_slam_visualization.launch.py
```

## ROS2 Topics

Published Topics:
- `/slam/odometry` - Odometry estimates
- `/slam/pose` - Current pose
- `/slam/trajectory` - Trajectory points
- `/slam/keyframes` - Extracted keyframes
- `/slam/statistics` - Performance statistics
